Tendril Wiki Help

Wiki Syntax

Tendril Wiki uses a custom wikitext syntax that is a simplified version of markdown. The scope is currently very limited, but may change to include more of the markdown syntax in the future.

The following sections will be examples of the currently supported syntax features.

Links

In addition to automatic link detection, Tendril Wiki has a special syntax for quickly linking to other notes in your notebook. This is done by using brackets, [[]] to surround the title of the note you want to link.

[[My Note]] will link to a page called, "My Note"

You can also alias note names to other text. This is to preserve the same behavior of HTML anchor tags where the href property and text content can differ.

For example, [[A cool note!|My Note]] will link to a page called, "My Note", with the link text displaying, "A cool note!"

[[files:article.pdf]] will link to an uploaded pdf called, "article"

[[this article|files:article.pdf]] will link to an uploaded pdf called, "article" with the link text displaying, "this article"

Quotes

To render a quote, you should start the line with > which may optionally be followed by a space. This means > this is a quote and >this is a quote will result in the same output.

Everything Else

All other text that you write in your notes will be interpreted as plaintext, with the sole exception of link detection.

Todo.txt Format

The task tracking functionality in Tendril Wiki follows the todo.txt format. Your tasks are stored in a todo.txt file inside your wiki directory.

Archiving Bookmarks

If you add the bookmark tag to a note in addition to adding url:<url-of-bookmark> to the metadata, tendril will archive the text content of the url. This text content will be availble inside the search engine, allowing you to do a full text search across the bookmarks you've saved!

Automatic Embeds

Certain websites and links to filetypes will automatically be embedded in your wiki by just pasting the link into the note!

Here some examples of sites and files that you can automatically embed:

  • Youtube videos
  • Photos copied to your clipboard
  • Spotify songs
  • Codepen pens
  • Codesandbox sandboxes
  • Vimeo videos
  • Links to mp3, ogg, and flac files

Keyboard Shortcuts

Keyboard shortcuts are enabled regardless of whether or not you are using JavaScript. If you aren't using JavaScript, the keyboard shortcuts are the same as the key listed below, in combination with your browser's access key combination

  • / Jump to the search page
  • n Create a new page
  • l Create a new page that backlinks to the currently viewed page
  • p Edit your user stylesheet
  • t View your task list
  • b Add a new bookmark
