Tendril Wiki Help

Wiki Syntax

In addition supporting GitHub flavored markdown, Tendril Wiki supports some wiki syntax extensions.

[[My Note]] will link to a page called, "My Note"

[[A cool note!|My Note]] will link to a page called, "My Note", with the link text displaying, "A cool note!"

[[files:article.pdf]] will link to an uploaded pdf called, "article"

[[this article|files:article.pdf]] will link to an uploaded pdf called, "article" with the link text displaying, "this article"

Todo.txt Format

The task tracking functionality in Tendril Wiki follows the todo.txt format. Your tasks are stored in a todo.txt file inside your wiki directory.

Archiving Bookmarks

If you add the `bookmark` tag to a note in addition to adding `url:` to the metadata, tendril will archive the text content of the url. This text content will be availble inside the search engine, allowing you to do a full text search across the bookmarks you've saved!

Automatic Embeds

Certain websites and links to filetypes will automatically be embedded in your wiki by just pasting the link into the note!

Here some examples of sites and files that you can automatically embed:

  • Youtube videos
  • Photos copied to your clipboard
  • Spotify songs
  • Codepen pens
  • Codesandbox sandboxes
  • Vimeo videos
  • Links to mp3, ogg, and flac files

Keyboard Shortcuts

Keyboard shortcuts are enabled regardless of whether or not you are using JavaScript. If you aren't using JavaScript, the keyboard shortcuts are the same as the key listed below, in combination with your browser's access key combination

  • / Jump to the search page
  • e Edit the currently viewed page
  • n Create a new page
  • l Create a new page that backlinks to the currently viewed page
  • p Edit your user stylesheet
  • t View your task list
